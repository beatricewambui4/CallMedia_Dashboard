# 📞 CALLMEDIA Call Centre Dashboard

## Overview
An interactive Power BI dashboard analysing 3.2M+ call centre records 
across 181 NYC agencies from February to July.

## Key Metrics
- **Total Calls:** 3,296,882
- **Total Agencies:** 181
- **Total Resolved Calls:** 1,738,693
- **Total Failed Calls:** 18,372

## Key Findings
- 52.74% of all calls were fully resolved at first contact
- Monday 11:00 AM is the peak call hour
- DOF has the highest number of failed/unresolved calls (9,369)
- 9 AM–3 PM accounts for bulk of daily call traffic
